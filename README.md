# 📝 To-Do List App

A simple and responsive To-Do List application built using **HTML, CSS, and JavaScript**.

## ✨ Features

* ➕ Add new tasks
* 🗑️ Delete tasks
* ✏️ Edit tasks
* 📋 Display tasks in a list
* 🎨 Clean and modern UI
* 📱 Responsive design
* ⚡ Built with Vanilla JavaScript

## 🛠️ Technologies Used

* 🌐 HTML5
* 🎨 CSS3
* ⚙️ JavaScript (ES6)
